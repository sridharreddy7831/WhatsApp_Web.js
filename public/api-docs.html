WhatsApp QR Bill API

Invoice generation, WhatsApp messaging, and payment QR codes

v1.0
Base URL: http://localhost:5000/api/v1

🔐 Authentication

All API endpoints require an API key. Include it in every request using one of these methods:

Header (recommended): X-API-Key: your_api_key_here

Query parameter: ?api_key=your_api_key_here

Rate limit: 100 requests/hour per key. Rate limit headers are included in every response.

⚡ Quick Start — Generate an API Key

Generate a new API key to get started. Admin endpoints are only accessible from localhost.

📄 Invoices
POST /api/v1/invoices Create a new GST Invoice ›

Request Body (JSON)

Field Type Description
customerName required string Customer's full name
customerPhone required string 10-digit phone number
items[] required array Array of line items
items[].name required string Item name
items[].qty required number Quantity
items[].price required number Unit price
items[].gstPercent number GST % (e.g. 18)
items[].hsnSac string HSN/SAC code
items[].uom string Unit of measure (KG, NOS, etc.)
paymentMode string Cash, Card, UPI, Bank Transfer
billTo object Billing address (name, address, phone, email, gstin)
shipTo object Shipping address (name, address)
dueDate string Payment due date (YYYY-MM-DD)
placeOfSupply string State/city of supply
notes string Additional notes
template string "gst" or "simple"

Example Request

{
  "customerName": "John Doe",
  "customerPhone": "9876543210",
  "items": [
    { "name": "Basmati Rice 5kg", "qty": 2, "price": 1750, "gstPercent": 5, "hsnSac": "1006", "uom": "KG" }
  ],
  "paymentMode": "UPI",
  "dueDate": "2026-03-01"
}

Response

{
  "success": true,
  "invoice": {
    "invoiceNumber": "INV-260213-027",
    "grandTotal": 3675,
    "amountInWords": "Three Thousand Six Hundred Seventy Five",
    ...
  }
}
GET /api/v1/invoices List invoices with pagination ›

Query Parameters

Param Type Default Description
page integer 1 Page number
limit integer 20 Items per page (max 100)
GET /api/v1/invoices/:id Get a single invoice by number ›

Path Parameters

Param Type Description
id required string Invoice number (e.g. INV-260213-027)
GET /api/v1/invoices/:id/pdf Download invoice as PDF ›

Path Parameters

Param Type Description
id required string Invoice number

Query Parameters

Param Type Default Description
format string a4 "a4" or "thermal"
template string gst "gst" or "simple"

Returns application/pdf binary. Use the browser or a PDF viewer to display.

💬 WhatsApp Messaging
POST /api/v1/send-message Send a text message via WhatsApp ›

Request Body (JSON)

Field Type Description
phone required string Phone number (10 digits, 91 prefix auto-added)
message required string Message text to send

Example Request

{
  "phone": "9876543210",
  "message": "Hello from the API!"
}

Response

{ "success": true, "phone": "9876543210", "messageSent": true }
POST /api/v1/send-invoice Send invoice PDF to customer's WhatsApp ›

Request Body (JSON)

Field Type Description
invoiceNumber required string Invoice number to send

Response

{ "success": true, "invoiceNumber": "INV-260213-027", "sentTo": "9876543210" }

Request Body (JSON)

Field Type Description
shopName string Shop display name
address string Full address
gstNumber string GSTIN
phone string Contact phone
email string Contact email
upiId string UPI payment ID

Only include fields you want to update. Others will retain existing values.

💳 Payments
POST /api/v1/generate-qr Generate UPI payment QR code ›

Request Body (JSON)

Field Type Description
upiId required string UPI ID (e.g. shop@paytm)
name string Payee name
amount number Amount to pay
note string Payment note

Response

{
  "success": true,
  "qrDataUrl": "data:image/png;base64,...",
  "upiString": "upi://pay?pa=shop@paytm&am=100&cu=INR"
}
⚠️ Error Codes
Status Error Description
400 Bad Request Missing or invalid parameters
401 Unauthorized Missing or invalid API key
403 Forbidden API key has been revoked
404 Not Found Invoice or resource not found
429 Too Many Requests Rate limit exceeded (check X-RateLimit-* headers)
503 Service Unavailable WhatsApp not connected
500 Server Error Internal error
🔧 cURL Examples

Create Invoice

curl -X POST http://localhost:5000/api/v1/invoices \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "customerName": "John Doe",
    "customerPhone": "9876543210",
    "items": [{"name": "Widget", "qty": 5, "price": 200, "gstPercent": 18}]
  }'

Download PDF

curl -H "X-API-Key: YOUR_API_KEY" \
  "http://localhost:5000/api/v1/invoices/INV-260213-027/pdf" \
  --output invoice.pdf

Send WhatsApp Message

curl -X POST http://localhost:5000/api/v1/send-message \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"phone": "9876543210", "message": "Hello from API!"}'
